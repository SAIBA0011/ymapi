# Yourmembership PHP Api

## Install Requirements

* Laravel
* Composer
* PHP 5.4 or above


Install via Composer:
```php
    "require": {
            "saiba/ymapi": "dev-master"
        },
```

Add YmapiServiceProvider to config/app.php:
```php
	'providers' => array(
            'Saiba\Ymapi\YmapiServiceProvider'
            );
```

Done
